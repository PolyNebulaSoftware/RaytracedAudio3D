# C# RaytracedAudio Player & Listener
A CSharp version of [Tienne_k's raytraced-audio](https://github.com/WhoStoleMyCoffee/raytraced-audio) addon for CSharp Godot projects. 

## Classes
- `RaytracedAudioListener3D` extends `AudioListener3D`
- `RaytracedAudioPlayer3D` extends `AudioStreamPlayer3D`

Use them as you would use `AudioListener3D` and `AudioStreamPlayer3D`.
